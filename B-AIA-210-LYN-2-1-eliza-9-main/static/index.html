Responsabilité

Conçu de façon
éthique

IONISBot intègre dès sa conception les principes d'un usage responsable de l'IA.

✅ Transparence
Le bot s'identifie comme un système automatique. Aucune tentative d'imiter un humain.
✅ Vie privée
Aucune donnée personnelle collectée. Historique en mémoire de session uniquement.
⚠️ Biais algorithmiques
Mistral peut présenter des biais culturels. Le prompt système cadre strictement les réponses.
✅ Désinformation
Données campus vérifiées. Température 0.2 pour minimiser les hallucinations du modèle.
✅ Dépendance
Le bot renvoie vers les services officiels pour les démarches importantes.
⚠️ Effet ELIZA
Langage volontairement fonctionnel, non émotionnel. Commande /reset disponible sur Discord.
